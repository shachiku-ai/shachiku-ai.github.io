Core Capabilities

Everything you need from your tireless AI assistant.

Single Binary Deployment
Frontend and backend compiled into one file via go:embed. No heavy Nginx or Node environments needed.
Server & CLI First
Robust for VPS/NAS logic with an integrated CLI to manage settings or trigger workflows right from your terminal.
Layered Memory
Short-term daily context managed by local SQLite, combined with long-term vectorized chromem storage.
Task Scheduler
Built-in Cron system automatically wakes the AI to execute periodic instructions silently.
Pluggable Skills
Easily extend AI capabilities: read APIs, run web crawlers, or process database records.
Telegram Integration
Native Watchers and notifications immediately push event updates to your Telegram client.
i18n & Multi-Model
English, Chinese, Japanese UI natively supported. Plugs into OpenAI, Gemini, and Claude seamlessly.
Automated HTTPS
Just drop your certificate files into the data folder and automatically bind to port 443 with TLS.

Built with Modern Stack

Reliable, fast and responsive design.
