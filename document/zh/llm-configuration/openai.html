OpenAI 访问配置

Shachiku 完美支持 OpenAI 全系列的大语言模型（如 gpt-4o 等）。下面是获取 API Token 并配置该服务的详细指引：

获取 API Token (Key)

  1. 访问 OpenAI 开发者平台 并登录您的账号。
  2. 导航至左侧菜单栏的 API Keys 页面。
  3. 点击 Create new secret key 按钮，您可以为该 Key 命名（例如 Shachiku），以便之后管理。
  4. 创建成功后，复制这段以 sk- 开头的密钥（请注意：密钥仅显示一次，请务必将其妥善保管）。

在系统中配置

  1. 首次启动 Shachiku 时，若系统检测到尚未配置大模型，您会被引导进入**初始化配置(Setup)**页面，或者您可以通过后管设置界面进入。
  2. 在 Provider 选项中选择 OpenAI。
  3. 将刚才复制的 API Key 填入对应的 API Key 输入框内中。
  4. (可选) 如果您需要使用代理服务或第三方兼容 OpenAI 通信协议的接口，请在开启服务前通过配置对应的 Base URL 指向代理地址（默认对接官方地址）。
  5. 点击操作界面上的验证与保存即可完成配置。系统会自动拉取所开放的模型列表让您选取主模型。
