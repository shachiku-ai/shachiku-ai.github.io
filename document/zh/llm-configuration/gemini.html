Google Gemini 配置指南

想要直连谷歌体系下的最强推理，并且享受优秀的上下文支持，您可以配置我们的 Gemini 连接通道。

获取 API Token (Key)

  1. 确保您已登录您的 Google 账号，然后访问 Google AI Studio。
  2. 在工作室的主界面左侧菜单中，寻找并点击 Get API Key。
  3. 在随后展现的页面内点击 Create API Key。如果在向导弹窗中碰到提示，您可以选择把它挂载建立在一个新项目或者既有的 Google Cloud Platform (GCP) 项目之中。
  4. Key 成功建立后，系统将展示一条字符串格式的 API 密钥将其完整复制并保存。

在系统中配置

  1. 返回 Shachiku 的前台配置引导流程或者设置选项板中。
  2. 在支持厂商列表里找到并将 Provider 切换至 Gemini (或 Google)。
  3. 在下方的 API Key 一栏快速填入并粘贴您获得的这串字符。
  4. 服务状态被无误保存后，您可以直接在应用界面内选择运行的默认 AI 模型（例如 gemini-1.5-pro）。
