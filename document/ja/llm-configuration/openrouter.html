OpenRouter アクセス設定

Shachikuは、世界中の多くのトップクラスの大規模言語モデル（anthropic/claude-3.5-sonnet、meta-llama/llama-3.1-405b など）を集めたユニファイドインターフェースプラットフォームであるOpenRouterを完璧にサポートしています。以下は、API Tokenを取得し、サービスを設定するための詳細なガイドです：

API トークン (Key) の取得

  1. OpenRouter 公式サイト にアクセスし、アカウントにログインまたは登録します。
  2. Keys ページに移動します。
  3. Create Key ボタンをクリックします。後で管理しやすいように、このKeyに名前（例：Shachiku）を付けることができます。
  4. 作成に成功したら、作成されたAPI Key（sk-or-v1- で始まる）をコピーします（注意：キーは一度しか表示されないため、必ず大切に保管してください）。

システムでの設定

  1. Shachikuの初回起動時に、大規模モデルが設定されていないことをシステムが検出すると、初期設定 (Setup) ページに案内されます。またはバックエンドの設定インターフェースから入ることもできます。
  2. Provider オプションで OpenRouter を選択します。
  3. 先ほどコピーしたAPI Keyを対応する API Key 入力ボックスに入力します。
  4. 操作インターフェースで「検証と保存」をクリックして設定を完了します。システムは自動的に公開されているモデルリストを取得し、メインモデルを選択できるようにします。
